# ONLIFIT UI RULES

## RESPONSIVE RULES

- Mobile-first design
- No horizontal scrolling
- No overlapping content
- No hidden content behind navbar
- Responsive containers only
- Avoid fixed widths on mobile

---

# SPACING SYSTEM

Use consistent:
- page padding
- section spacing
- card padding
- form spacing

---

# DASHBOARD RULES

Dashboards should:
- feel like premium SaaS
- use shared dashboard shell
- use shared navbar system
- use shared spacing utilities

---

# COMPONENT RULES

Reusable globally:
- trainer cards
- buttons
- forms
- dashboard cards
- tabs
- navbar
- sidebar

---

# DESIGN RULES

- Preserve branding
- Preserve functionality
- Avoid redesigning layouts unnecessarily
- Improve stability first
- Improve consistency second
